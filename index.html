What is JavaScript

"JavaScript is a cross-platform, object-oriented scripting language. JavaScript is a small, lightweight language; it is not useful as a standalone language, but is designed for easy embedding in other products and applications, such as web browsers. Inside a host environment, JavaScript can be connected to the objects of its environment to provide programmatic control over them."

-Mozilla Developer Network

Extending JavaScript

Client-side JavaScript extends the core language by supplying objects to control a browser (Navigator or another web browser) and its Document Object Model (DOM).

Server-side JavaScript extends the core language by supplying objects relevant to running JavaScript on a server.

-Mozilla Developer Network

The Noscript Element

This element is used when browsers the does not support JavaScript. Any content inside contained in the noscript element will be displayed in the following circumstances:

  • The browser does not support JS Scripting
  • The browser's scripting support has been turned off

Comments

JavaScript uses C-style comments for both single-line and block comments

A single-line comment begins with two forward-slash charaters


//single line comment

Block comment begins with a forward slash and asterick ( /* )


/*
* This is a multi-line 
* comment
*/

Always comment your code..

Strict Mode

ECMAScript 5 introduced the concept of Strict Mode. To Enable strict mode for an entire script, include the following at the top: 


					"use strict";

You can also specify just a function to execute in strict mode by including the pragma at the top of the function body:



function doSomthing(){
	"use strict";
	//function body
}

Statements

Statements in JavaScript are terminated by a semicolon. 


var sum = a + b     // valid even without the semicolon  - not recomended
var diff = a - b;   // valid - preferred

Muliple statements can be combined into a code block by using syntax beginning with a left curly brace ( { ) and ending with a right curly brace ( } ).


if (test){
  test = false;
  alert(test);
}

Control Statements

Control Statements, such as (if) require code blocks only when execution multiple statements


if (test)
  alert(test);   //valid but error-prone and should be avoided


if (test) {
  alert(test);  //preferred
}

Using code blocks for control statements make the intent clear and less chance for errors when changes are made.

Keyword and Reserved words

Keyword has special meaning in the JavaScript language and are part of the syntax

Reserved words are word that cannot be used as indentifiers (variables, functions, etc.) because that are reserved by the language.
